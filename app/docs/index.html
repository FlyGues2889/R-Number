文档

本项目是使用 Electron 和 MDUI 制作的随机数抽取器。

  • Gitee Releases

开发

本项目使用 electron 和 electron-packager 开发，请确保您的电脑上已安装 Node.js 环境和对应依赖。

项目 Gitee 地址 项目 GitHub 地址

克隆项目代码：

git clone https://gitee.com/flygues2889/random-number.git

（GitHub 上的项目代码已不再更新。GitHub上仅提供v0.2.0以前的版本） 

git clone https://github.com/FlyGues2889/random-number.git

安装依赖：

cnpm install

随后请定位到项目内app/目录下，此为项目所在目录。

运行项目：

npm start

打包安装包：

electron-packager .

使用

打开软件后，点击主页上的“开始”按钮，即可开始随机数抽取。软件默认抽取范围为 1 到 55 之间的数字，所以会随机生成一个 1 到 55 之间的数字，并显示在屏幕上。

  • 点击 开始抽取 按钮 / 回车键 / 空格键 开始抽取数字。
  • 在 历史记录 中查看已抽取的数字并复制。
  • 在 设置 中设置抽取数字范围、抽取时长、是否重复和是否手动抽取。
