RandomNumber

使用 Electron 和 MDUI 制作的随机数抽取器

下载 在线示例

文档

本项目是使用 Electron 和 MDUI 制作的随机数抽取器。

用户版本安装

本项目提供用户版本安装，可以直接下载安装。点击下方按钮或链接以下载安装包。

下载安装包 Win32 x64 版本

可供下载的地址：

  • Github Releases
  • Gitee Releases

或者使用在线示例版本：

在线示例版本

开发

本项目使用 electron 和 electron-packager 开发，请确保您的电脑上已安装 Node.js 环境和对应依赖。

项目 GitHub 地址
项目 Gitee 地址

克隆项目代码：

git clone https://gitee.com/flygues2889/random-number.git
git clone https://github.com/FlyGues2889/random-number.git

安装依赖：

cnpm install

随后请定位到项目内app/目录下，此为项目所在目录。

运行项目：

npm start

打包安装包：

electron-packager .

使用

主页

  • 点击 抽取 按钮，随机生成指定范围内的随机数。
  • 在已抽取的数字上按下 鼠标右键 ，可以打开复制菜单复制随机数。
  • 点击 全屏 按钮可以打开/关闭全屏模式。
  • 点击 使用指南 按钮，可以查看使用指南。

记录页

  • 点击 复制记录 按钮可以复制已抽取到的数字记录。
